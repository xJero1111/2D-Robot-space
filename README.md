# 2D_Robot_space
¡Juego Top-Down sobre un robot espacial que recolecta energia!
2D Robot Space
Este es un juego de exploración corta desarrollado en Unity para la clase de lenguajes de programación. El jugador controla un pequeño dron que debe decidir el futuro de una nave varada en el espacio.

Cómo jugar
Movimiento: Teclas WASD o Flechas.

Objetivo: Recolecta 10 energías del mismo tipo (Tecnológicas o Naturales) para activar uno de los dos finales positivos.

Peligro: Evita los cables con chispas; tocar uno activará el "Final Gris" (pérdida).

Reiniciar: Presiona R cuando termine la partida para volver a intentarlo.

Notas de desarrollo
El proyecto utiliza un sistema de llaves por ID para abrir puertas y lógica de estados en el GameManager para gestionar los tres finales posibles. Desarrollado como parte del proceso de Diseño Interactivo en EAFIT.
